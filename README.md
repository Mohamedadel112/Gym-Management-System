## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Recommended version 18+)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- (Optional) [Git](https://git-scm.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mohamedadel112/Gym-Management-System.git
   ```
2. **Navigate to the project directory**
   ```bash
   cd Gym-Management-System
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```

---

## How to Run

### Development Mode

```bash
npm run dev
```

This will start the application with hot-reload in development mode.

### Production Build

```bash
npm run build
npm start
```

This will build the project and then start the server.

### Alternative (If your project is Frontend-only)

If your project only provides static HTML (and compiled JS) files, open the `index.html` file in your browser **after building** (`dist` or `build` folder).

Or, serve the `dist` folder with a static server:
```bash
npm install -g serve
serve -s dist
```

## Usage

- Access the running app in your browser at [http://localhost:3000](http://localhost:3000) (or the port specified in your config).
- Log in or register as instructed.
- Manage gym members, trainers, schedules, classes, and more.
